# 🎙️ Tamil Voice Recharge Assistant

A voice-based web application designed for rural users to perform mobile recharge and bill payments using simple Tamil voice commands.

---

## 🎯 Project Goal
To help low-literacy rural users complete digital transactions easily using voice guidance in Tamil.

---

## 🚀 Features
- 🗣️ Full Tamil Voice Guidance (Step-by-step)
- 🎧 Speech Recognition (Voice Input)
- 🔊 Text-to-Speech (Human-like Tamil Voice)
- 📱 Mobile Recharge, DTH, Electricity, Gas Payments
- ✅ Real-time Validation & Error Handling
- 📜 Voice Transcript Display
- 👆 Simple & Large UI for easy usage

---

## 🧠 AI Usage
- Speech-to-Text using Web Speech API
- Text-to-Speech using ElevenLabs API
- Smart voice interaction & validation system

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Web Speech API
- ElevenLabs API
- Google Fonts (Noto Sans Tamil)

---

## 📱 Application Flow
1. Voice Welcome
2. Service Selection (Voice/Button)
3. Input Details (Voice + Manual)
4. Validation
5. Plan Selection
6. Confirmation
7. Success Screen

---

## 🔐 Validation Rules
- Mobile: 10 digits (6–9 start)
- DTH: 6–12 digits
- Electricity: 8–15 digits
- Gas: 10–17 digits
- OTP: 4 digits (Demo: 1234)

---

## 🎯 Target Users
Rural women with low literacy and minimal technical knowledge.

---

## 🌟 Key Benefit
Removes the need for reading, typing, or English understanding.
